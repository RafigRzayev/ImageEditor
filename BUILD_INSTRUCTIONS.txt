1) Download the repository.

2) Install SDL2 on your system( sudo apt-get install libsdl2-dev)

3) Open src folder in the terminal.

4) Run following commands with gcc compiler:
g++ -o main -Wall -pedantic main.cpp image.cpp buffer.cpp display.cpp -lSDL2 process_user_commands.cpp helper_functions.cpp IP_change_brightness.cpp IP_extract_channel.cpp IP_histogram.cpp IP_monochrome.cpp IP_quantize.cpp IP_rotate.cpp -O3

5) Start program by typing ./main in the terminal
