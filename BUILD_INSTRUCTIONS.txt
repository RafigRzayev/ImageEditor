1) Go to the terminal

2) Type "sudo apt install cmake libsdl2-dev g++ git" to download the dependencies

3) Type "git clone https://github.com/RafigRzayev/ImageEditor" to download the repository

3) Go to the "build" folder

4) Type "cmake ../src", afterwards type "make"

6) Start program by typing "./ImageEditor"

7) Use the command line to interface with the program
     You can start by loading the default image: 
     - Select "Load" option
     - Type "../gallery/default.jpg"
